This folder contains daily soil moisture probe data supplied from the SASMAS project covering sites in the Upper Hunter Valley, Australia (http://www.eng.newcastle.edu.au/sasmas/SASMAS/sasdata.html)

The data file name relates to the depths over which the observation cover. 

Please acknowledge the SASMAS project along with the following reference:
Rüdiger, C., Hancock, G., Hemakumara, H.M., Jacobs, B., Kalma, J.D., Martinez, C., Thyer, M., Walker, J.P., Wells, T. and Willgoose, G.R., 2007. Goulburn River experimental catchment data set. Water Resources Research, 43(10): W10403.
